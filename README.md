dp-code-list-api

An API used to navigate code lists

### Getting started
      
### Configuration
    
| Environment variable | Default                                   | Description
| -------------------- | ----------------------------------------- | -----------
| BIND_ADDR            | :21800                                    | The host and port to bind to
| MONGODB_ADDR | "localhost:27017"                         | A URL to mongodb
